# College-Enquiry-Chatbot
The College Enquiry Chatbot is an AI-based automated system designed to provide instant and accurate information to students, parents, and visitors regarding college-related queries. It helps users get details about admissions, courses, fees structure, eligibility, placement, scholarships, hostel facilities, events, and contact 
 Features

-  AI-powered chatbot for instant responses  
-  24/7 availability  
- College information (About, Courses, Departments)  
- Admission process & eligibility details  
-  Fee structure & scholarship information  
- Hostel & campus facilities  
-  Placement & career guidance details    
-  User-friendly interface  

 Objectives

- To automate college enquiry handling  
- To provide quick and accurate information to users  
- To reduce administrative workload  
- To enhance user experience during admission periods  

---
 Technology Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python / Node.js  
- **Framework:** Flask  
- **Database:** SQLite  
- **AI/NLP:** Rule-based / NLP-based chatbot  
 

 System Architecture

1. User sends a query via chatbot interface  
2. Chatbot processes input using predefined rules / NLP  
3. Relevant response is fetched from database  
4. Response is displayed to the user instantly
